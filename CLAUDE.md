# high.school_subject — Claude Code 專案指南

## 專案概述

高中學科筆記與學習資源整理，涵蓋台灣高中 13 個主要學科。

---

## 專案結構

```
high.school_subject/
├── 國文/          # 語文、文學、作文
├── 英文/          # 文法、閱讀、寫作
├── 數學/          # 代數、幾何、微積分
├── 物理/          # 力學、電磁、光學
├── 化學/          # 有機、無機、化學反應
├── 生物/          # 細胞、遺傳、生態
├── 地球科學/      # 大氣、地質、天文
├── 歷史/          # 台灣史、中國史、世界史
├── 地理/          # 自然地理、人文地理
├── 公民與社會/    # 政治、法律、經濟
├── 體育/          # 運動理論與實作
├── 藝術/          # 音樂、美術、藝術史
└── 資訊科技/      # 程式設計、資料結構
```

---

## 檔案命名規則

- 每個學科根目錄放 `README.md`（學科總覽）
- 筆記檔案命名：`ch1.md`, `ch2.md`...
- 考題整理：`exam.md`
- 重點複習：`review.md`

---

## Git 工作流程

- 主分支：`main`
- 功能分支命名：`claude/feature-name`
- 每個功能開獨立 PR，不直接 push main
