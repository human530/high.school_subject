# 英文學習 App — MVP

針對升高一學生、目標學測 15 級分的個人化英文拼字練習工具。

## 使用方式

直接用瀏覽器打開 `index.html` 即可使用,不需要任何後端或安裝。

```bash
# 本機開啟(macOS)
open index.html

# 本機開啟(Linux)
xdg-open index.html

# 或用 Python 開簡單伺服器
python3 -m http.server 8000
# 然後瀏覽器打開 http://localhost:8000
```

## 檔案結構

```
learning-app/
├── index.html   # 主頁面(練習 / 統計 / 錯題本 三個分頁)
├── style.css    # 深色模式 + 響應式樣式
├── app.js       # SRS 演算法 + 練習邏輯
├── words.js     # 50 個學測常用單字
└── README.md    # 本檔
```

## SRS 間隔重複規則

| Level | 名稱       | 下次出現 |
|-------|-----------|---------|
| 1     | 剛學       | 當天    |
| 2     | 熟悉       | 1 天後  |
| 3     | 熟練       | 3 天後  |
| 4     | 精通       | 7 天後  |
| 5     | 長期記憶   | 30 天後 |

- 答對 → Level +1 (最高 5)
- 答錯 → Level 歸 1,當天再考一次
