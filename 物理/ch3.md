# 第三章：天體運動

## 3-3 克卜勒行星運動定律

---

### 橢圓補充知識

**橢圓的定義：**
平面上到兩定點（焦點）之距離和為一定值的點所成之集合。

**橢圓基本參數：**
| 符號 | 名稱 | 說明 |
|------|------|------|
| a | 半長軸 | 長軸的一半 |
| b | 半短軸 | 短軸的一半 |
| c | 焦距 | 中心到焦點的距離 |
| e | 離心率 | e = c/a |

**重要關係式：**
$$a^2 = b^2 + c^2$$
$$r_1 + r_2 = 2a \quad \text{（橢圓上任意點到兩焦點距離之和為定值）}$$

**畫橢圓的方法：**
用一條繩子固定在橢圓內的兩焦點（O₁、O₂），以筆拉緊繩子滑動，即可畫出橢圓。

---

### 克卜勒第一定律（橢圓定律）

- 行星繞太陽公轉的軌道為**橢圓形**
- **太陽位於橢圓的一個焦點**
- 行星與太陽的距離（日距）會隨位置改變

**近日點與遠日點：**
$$r_{\min} = a - c \quad \text{（近日點，行星距太陽最近）}$$
$$r_{\max} = a + c \quad \text{（遠日點，行星距太陽最遠）}$$

**日距的算術平均值等於半長軸 a：**
$$\frac{r_{\min} + r_{\max}}{2} = \frac{(a-c)+(a+c)}{2} = a$$

**推導（利用橢圓性質）：**
- 當行星 P 滑動到端點 R 時，r₁ = a − c，RO₂ = a
- 根據 r₁ + r₂ = 定值，可知 RO₁ = RO₂ = a
- 故橢圓內三段長度關係為：$a^2 = b^2 + c^2$

---

### 重點整理

1. **橢圓上任一點**到兩焦點距離之和 = 2a（定值）
2. **近日點** r_min = a − c；**遠日點** r_max = a + c
3. **日距算術平均** = 半長軸 a
4. **三邊關係**：a² = b² + c²
5. 太陽位於橢圓**焦點**，不在中心
