# 數A 第三冊 — 三角函數重點與公式（完整版）

> 涵蓋範圍：廣義角、弧度制、三角函數定義、基本關係、誘導公式、和差角、倍角半角、萬能公式、和差化積、積化和差、正餘弦定理、投影定理、中線長、三角形面積、三角函數圖形、輔助角、反三角函數。

---

## 一、角的度量

### 1. 廣義角
- 角度有正負：逆時針為正角，順時針為負角。
- 同界角（終邊相同的角）：θ + 360°·n（n ∈ ℤ）或 θ + 2nπ。

### 2. 弧度制（徑度量）
- 定義：1 弧度 = 半徑長的弧所對的圓心角。
- 換算：
  - π (rad) = 180°
  - 1° = π/180 (rad)
  - 1 (rad) = 180°/π ≈ 57.3°

### 3. 弧長與扇形面積
設圓半徑 r，圓心角 θ（弧度）：
- 弧長：**s = rθ**
- 扇形面積：**A = (1/2) r²θ = (1/2) r·s**

## 二、三角函數的定義

### 1. 廣義角定義（坐標定義）
設 P(x, y) 為角 θ 終邊上一點，r = √(x² + y²)：

| 函數 | 定義 |
|------|------|
| sin θ | y / r |
| cos θ | x / r |
| tan θ | y / x （x ≠ 0） |
| cot θ | x / y （y ≠ 0） |
| sec θ | r / x （x ≠ 0） |
| csc θ | r / y （y ≠ 0） |

### 2. 各象限正負號（ASTC 法則「全正、S、T、C」）
| 象限 | sin | cos | tan | 為正者 |
|------|-----|-----|-----|--------|
| I    |  +  |  +  |  +  | All 全正 |
| II   |  +  |  −  |  −  | Sin     |
| III  |  −  |  −  |  +  | Tan     |
| IV   |  −  |  +  |  −  | Cos     |

## 三、特殊角三角函數值

| θ       | 0  | π/6   | π/4   | π/3   | π/2 |
|---------|----|-------|-------|-------|-----|
| sin θ   | 0  | 1/2   | √2/2  | √3/2  | 1   |

| tan θ   | 0  | √3/3  | 1     | √3    | 不存在 |

**記憶法**：sin 依序為 √0/2, √1/2, √2/2, √3/2, √4/2；cos 反序。

### 2. 擴充至一整圈

## 四、基本關係式

### 1. 平方關係
- **sin²θ + cos²θ = 1**
- **1 + tan²θ = sec²θ**
- **1 + cot²θ = csc²θ**

### 2. 商數關係
- tan θ = sin θ / cos θ
- cot θ = cos θ / sin θ

### 3. 倒數關係
- sin θ · csc θ = 1
- cos θ · sec θ = 1
- tan θ · cot θ = 1

## 五、誘導公式（化簡公式）完整表

口訣：**「奇變偶不變，符號看象限。」**
（π/2 的奇數倍：函數名 sin↔cos、tan↔cot、sec↔csc；偶數倍不變；符號由「原本 θ 視為銳角時新角所在象限」決定。）

| 角 | sin | cos | tan | cot |
|----|-----|-----|-----|-----|
| −θ          | −sin θ |  cos θ | −tan θ | −cot θ |
| π/2 − θ     |  cos θ |  sin θ |  cot θ |  tan θ |
| π/2 + θ     |  cos θ | −sin θ | −cot θ | −tan θ |
| π − θ       |  sin θ | −cos θ | −tan θ | −cot θ |
| π + θ       | −sin θ | −cos θ |  tan θ |  cot θ |
| 3π/2 − θ    | −cos θ | −sin θ |  cot θ |  tan θ |
| 3π/2 + θ    | −cos θ |  sin θ | −cot θ | −tan θ |
| 2π − θ      | −sin θ |  cos θ | −tan θ | −cot θ |
| 2kπ + θ     |  sin θ |  cos θ |  tan θ |  cot θ |

**奇偶性**：
- sin、tan、cot、csc 為奇函數：f(−θ) = −f(θ)
- cos、sec 為偶函數：f(−θ) = f(θ)

**週期性**：
- sin、cos、sec、csc 週期為 2π
- tan、cot 週期為 π

## 六、和角與差角公式

### 1. 正弦
- **sin(α + β) = sin α cos β + cos α sin β**
- **sin(α − β) = sin α cos β − cos α sin β**

### 2. 餘弦
- **cos(α + β) = cos α cos β − sin α sin β**
- **cos(α − β) = cos α cos β + sin α sin β**

### 3. 正切
- **tan(α + β) = (tan α + tan β) / (1 − tan α tan β)**
- **tan(α − β) = (tan α − tan β) / (1 + tan α tan β)**

### 4. 餘切
- cot(α + β) = (cot α cot β − 1) / (cot α + cot β)
- cot(α − β) = (cot α cot β + 1) / (cot β − cot α)

### 5. 記憶技巧
- sin：**「sin cos + cos sin」** ，正負號與主角同
- cos：**「cos cos − sin sin」** ，正負號與主角相反
- tan：**「同加異減」**，分母正負與分子相反

---

## 七、倍角公式

### 1. 二倍角
- **sin 2θ = 2 sin θ cos θ**
- **cos 2θ = cos²θ − sin²θ = 1 − 2 sin²θ = 2 cos²θ − 1**
- **tan 2θ = 2 tan θ / (1 − tan²θ)**
- cot 2θ = (cot²θ − 1) / (2 cot θ)
- sec 2θ = sec²θ / (2 − sec²θ)
- csc 2θ = (sec θ · csc θ) / 2

### 2. 三倍角
- **sin 3θ = 3 sin θ − 4 sin³θ**
- **cos 3θ = 4 cos³θ − 3 cos θ**
- **tan 3θ = (3 tan θ − tan³θ) / (1 − 3 tan²θ)**

### 3. 萬能公式（以 t = tan(θ/2) 表示）
設 t = tan(θ/2)：
- **sin θ = 2t / (1 + t²)**
- **cos θ = (1 − t²) / (1 + t²)**
- **tan θ = 2t / (1 − t²)**

常用於將 sin、cos 全部改寫成 tan(θ/2) 的代換。

---

## 八、半角公式

由 cos 2θ 的變形推得（θ/2 所在象限決定正負號）：
- **sin(θ/2) = ±√((1 − cos θ) / 2)**
- **cos(θ/2) = ±√((1 + cos θ) / 2)**
- **tan(θ/2) = ±√((1 − cos θ) / (1 + cos θ))**
  - = sin θ / (1 + cos θ)
  - = (1 − cos θ) / sin θ

**降冪公式**（消除平方，常用於積分）：
- **sin²θ = (1 − cos 2θ) / 2**
- **cos²θ = (1 + cos 2θ) / 2**
- **tan²θ = (1 − cos 2θ) / (1 + cos 2θ)**
- sin³θ = (3 sin θ − sin 3θ) / 4
- cos³θ = (3 cos θ + cos 3θ) / 4

## 九、和差化積與積化和差

### 1. 積化和差
- sin α cos β = (1/2)[sin(α+β) + sin(α−β)]
- cos α sin β = (1/2)[sin(α+β) − sin(α−β)]
- cos α cos β = (1/2)[cos(α+β) + cos(α−β)]
- sin α sin β = −(1/2)[cos(α+β) − cos(α−β)]

### 2. 和差化積
- sin A + sin B = 2 sin((A+B)/2) cos((A−B)/2)
- sin A − sin B = 2 cos((A+B)/2) sin((A−B)/2)
- cos A + cos B = 2 cos((A+B)/2) cos((A−B)/2)
- cos A − cos B = −2 sin((A+B)/2) sin((A−B)/2)

## 十、正弦定理與餘弦定理

設三角形 ABC，三邊為 a、b、c（對角為 A、B、C），外接圓半徑 R：

### 1. 正弦定理
**a / sin A = b / sin B = c / sin C = 2R**

**應用**：
- 已知兩角與一邊 (AAS / ASA)
- 已知兩邊與一對角 (SSA，可能有兩解)

### 2. 餘弦定理
- **a² = b² + c² − 2bc cos A**
- **b² = a² + c² − 2ac cos B**
- **c² = a² + b² − 2ab cos C**

變形求角：
- cos A = (b² + c² − a²) / (2bc)
- cos B = (a² + c² − b²) / (2ac)
- cos C = (a² + b² − c²) / (2ab)

**應用**：
- 已知三邊 (SSS)
- 已知兩邊及其夾角 (SAS)

### 3. 投影定理
三角形中任一邊等於其他兩邊在該邊上的投影之和：
- **a = b cos C + c cos B**
- **b = a cos C + c cos A**
- **c = a cos B + b cos A**

### 4. 中線長公式（Apollonius 定理）
設 m_a 為從 A 到 BC 中點的中線長：
- **m_a² = (2b² + 2c² − a²) / 4**
- **m_b² = (2a² + 2c² − b²) / 4**
- **m_c² = (2a² + 2b² − c²) / 4**

### 5. 角平分線長公式
設 t_a 為 ∠A 的角平分線長：
- t_a² = bc[1 − (a/(b+c))²]
- t_a = (2bc cos(A/2)) / (b + c)

### 6. 正切定理
- (a − b) / (a + b) = tan((A − B)/2) / tan((A + B)/2)

## 十一、三角形面積公式

設三角形 ABC，三邊 a、b、c，半周長 s = (a+b+c)/2，外接圓半徑 R，內切圓半徑 r：

1. **底乘高**：
   - △ = (1/2) × 底 × 高

2. **兩邊夾角**：
   - **△ = (1/2) ab sin C = (1/2) bc sin A = (1/2) ac sin B**

3. **海龍公式 (Heron)**：
   - **△ = √(s(s−a)(s−b)(s−c))**

4. **外接圓半徑**：
   - **△ = abc / (4R)**
   - 推得：**R = abc / (4△)**

5. **內切圓半徑**：
   - **△ = rs**
   - 推得：**r = △ / s**

6. **兩角一邊**：
   - △ = (a² sin B sin C) / (2 sin A)

7. **座標公式**：若 A(x₁,y₁)、B(x₂,y₂)、C(x₃,y₃)：
   - △ = (1/2)|x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)|
   - 行列式寫法：△ = (1/2)|det[[x₂−x₁, y₂−y₁],[x₃−x₁, y₃−y₁]]|

8. **向量公式**：若 u⃗ = AB⃗、v⃗ = AC⃗：
   - △ = (1/2)|u⃗ × v⃗|

### 內切圓、外接圓、旁切圓半徑補充
- R = a / (2 sin A) = b / (2 sin B) = c / (2 sin C)
- r = (s − a) tan(A/2) = (s − b) tan(B/2) = (s − c) tan(C/2)
- r = 4R sin(A/2) sin(B/2) sin(C/2)
- 旁切圓半徑：r_a = △ / (s − a)，其餘同理

## 十二、三角函數的圖形

### 1. y = sin x
- 定義域：ℝ
- 值域：[−1, 1]
- 週期：2π
- 奇函數，關於原點對稱

### 2. y = cos x
- 定義域：ℝ
- 值域：[−1, 1]
- 週期：2π
- 偶函數，關於 y 軸對稱

### 3. y = tan x
- 定義域：x ≠ π/2 + nπ
- 值域：ℝ
- 週期：π
- 奇函數
- 鉛直漸近線：x = π/2 + nπ

### 4. 一般形式 y = A sin(Bx + C) + D
- **振幅**：|A|
- **週期**：2π / |B|
- **相位移**：−C / B
- **垂直平移**：D
- 對 y = A cos(Bx + C) + D 同理。

## 十三、重要技巧與常見題型

### 1. 輔助角公式（合成公式）
**a sin x + b cos x = √(a² + b²) · sin(x + φ)**
其中 tan φ = b / a（注意象限）。

- 最大值：√(a² + b²)
- 最小值：−√(a² + b²)

### 2. 三角方程與三角不等式
- 解時注意**主值範圍**與**週期**。
- 一般解寫法：
  - sin x = k → x = nπ + (−1)ⁿ · arcsin k
  - cos x = k → x = 2nπ ± arccos k
  - tan x = k → x = nπ + arctan k

### 3. 三角形內的恆等式
由 A + B + C = π 可得：
- sin(A+B) = sin C，cos(A+B) = −cos C，tan(A+B) = −tan C
- sin((A+B)/2) = cos(C/2)，cos((A+B)/2) = sin(C/2)
- **tan A + tan B + tan C = tan A · tan B · tan C**
- cot A cot B + cot B cot C + cot C cot A = 1
- sin 2A + sin 2B + sin 2C = 4 sin A sin B sin C
- cos 2A + cos 2B + cos 2C = −1 − 4 cos A cos B cos C
- sin²A + sin²B + sin²C = 2 + 2 cos A cos B cos C

### 4. 判斷三角形形狀
- 最大邊的對角若為銳角 ⟹ 銳角三角形
- 若 cos(最大角) = 0 ⟹ 直角三角形
- 若 cos(最大角) < 0 ⟹ 鈍角三角形

### 5. 反三角函數基本
- arcsin x：值域 [−π/2, π/2]
- arccos x：值域 [0, π]
- arctan x：值域 (−π/2, π/2)
- 恆等式：
  - sin(arcsin x) = x
  - arcsin x + arccos x = π/2
  - arctan x + arctan(1/x) = π/2  (x > 0)

## 學習建議

1. **先背特殊角**（0、30°、45°、60°、90°），再由誘導公式擴充。
2. **和角公式是核心**：倍角、半角、和差化積都由它推得。
3. **畫圖理解**三角函數的週期性與對稱性，比死背誘導公式更可靠。
4. **正餘弦定理配合面積公式**是解三角形題型的黃金組合。
5. 練習時要會**在弧度與角度間自由切換**。
