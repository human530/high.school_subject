# 數學

## 🎯 數A 滿級分複習 App

👉 **[開啟 App（數學/app）](./app/index.html)** ｜ [說明文件](./app/README.md)

一款針對學測數學A **滿級分** 的離線複習 PWA，涵蓋 1～4 冊：白話教學＋秒殺解法、無限參數化練習、學測/段考模擬考（大考中心級分換算）、個人化弱點分析，並支援貼自己金鑰用 LLM 出題。

## 學習重點
- 代數與函數
- 幾何（平面、空間）
- 機率與統計
- 微積分基礎
