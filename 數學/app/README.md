# 📐 數A 滿級分複習 App

一款為 **學測數學A 滿級分** 設計的離線複習 PWA，涵蓋 108 課綱數A **第 1～4 冊**。
核心理念：**用 5 歲也聽得懂的白話 + 各章秒殺解法**，把你教到「看到題目就秒殺、正確率 90%+」，並用個人化分析補強弱點。

## ✨ 功能

| 功能 | 說明 |
|------|------|
| 📖 **白話教學** | 每章「5 歲也懂版」概念、核心觀念、**秒殺解法/口訣**、重點公式 |
| 📝 **經典題＋學測風格題** | 每章皆附經典題與學測風格題，含提示與完整白話詳解 |
| ✏️ **無限練習（引擎①）** | 離線**參數化題庫**：同題型自動換數字，無限練到精熟，附詳解 |
| 🤖 **LLM 出題（引擎②）** | 貼上**自己的 API 金鑰**呼叫真 LLM 出全新題（OpenAI 相容 / Anthropic）；金鑰只存本機 |
| 🧠 **模擬考** | **學測版**（全範圍 1~4 冊）＋**段考版**（自選範圍），內建計時器 |
| 🎯 **大考中心級分** | 依大考中心級分制度近似換算（滿 15 級），附五標解讀與「衝滿級還差幾分」 |
| 📊 **個人化分析** | 記錄每章對錯 → 算各章精熟度 → 自動排「邁向滿級分」補強優先序 |
| 📱 **PWA 離線可用** | 可加入主畫面、離線使用；資料只存在你的瀏覽器 |

## 🚀 使用方式

### 方法 A：直接雙擊（最簡單）
直接用瀏覽器開啟 `index.html` 即可使用全部離線功能（教學、練習、模擬考、分析）。
> 註：以 `file://` 開啟時 PWA 離線快取會自動略過，功能不受影響。

### 方法 B：本機伺服器（完整 PWA）
```bash
cd 數學/app
python3 -m http.server 8000
# 瀏覽器開 http://localhost:8000
```

### 方法 C：GitHub Pages（上線分享）
將 `數學/app` 設為 Pages 來源，即可手機/電腦透過網址使用並「加入主畫面」。

## 🧩 雙引擎出題

- **引擎①（預設、免金鑰、免網路）**：`js/generator.js` 的參數化題庫，每章對應數個 generator，數字隨機但答案簡潔，可無限出題。
- **引擎②（選用）**：到「設定」貼上你自己的 API 金鑰即可呼叫真 LLM 出全新題。金鑰僅存於瀏覽器 `localStorage`，不會上傳任何伺服器。

## 📚 內容範圍（108 課綱・數A 必修）

- **第一冊**：數與式（含絕對值/不等式）、多項式函數、指數與對數函數
- **第二冊**：數列與級數、排列與組合、機率（含條件機率/貝氏）、數據分析
- **第三冊**：三角比與正餘弦定理、三角函數（弧度/和差角/疊合）、直線與圓、平面向量
- **第四冊**：空間向量、空間中的平面與直線、矩陣

## 🗂 檔案結構

```
數學/app/
├── index.html              # 入口
├── manifest.webmanifest    # PWA 設定
├── sw.js                   # service worker（離線快取）
├── css/style.css
├── data/                   # 課程內容（資料與程式分離，好擴充）
│   ├── curriculum.js       # 容器與工具
│   └── book1~4.js          # 各冊教學/公式/秒殺/題目
└── js/
    ├── generator.js        # 引擎①：參數化題庫
    ├── llm.js              # 引擎②：真 LLM 出題
    ├── scoring.js          # 大考中心級分換算
    ├── analytics.js        # 個人化弱點分析
    ├── exam.js             # 學測/段考組卷與批改
    └── app.js              # 路由與 UI
```

## 🔧 擴充內容

要加題目或新章節，只需編輯 `data/book*.js`（純資料），或在 `generator.js` 新增 generator 函式並在章節的 `generators` 陣列引用，無需動到 UI。

## ⚠️ 注意

- 級分換算為**近似**（模擬考無真實母群），以「滿分對應 15 級」估算，僅供自我定位。
- 教學提供的「秒殺解法」以正確、好懂、好記為主，搭配課本正規解法效果最佳。
