# high.school_subject

高中學科筆記與學習資源整理。

## 電腦版使用說明

Clone 後如果中文資料夾名稱顯示為亂碼，請執行：

```bash
git config --global core.quotePath false
```

Windows 使用者建議額外設定：

```bash
git config --global core.autocrlf true
git config --global i18n.logOutputEncoding utf-8
git config --global i18n.commitEncoding utf-8
```
